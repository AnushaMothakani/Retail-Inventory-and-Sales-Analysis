# Retail-Inventory-and-Sales-Analysis

## Project Overview
Retail Inventory and Sales Analysis project developed using Excel, MySQL, and Power BI to analyze sales performance, inventory trends, regional sales insights, and business KPIs through interactive dashboards and data-driven reporting.

## Tools Used
- Excel
- MySQL
- Power BI

## Project Steps
1. Data Cleaning in Excel
2. Data Storage and Analysis using MySQL
3. Dashboard Creation in Power BI

## Key Insights
- Top selling products
- Monthly sales trends
- Regional performance
- Inventory risk analysis
